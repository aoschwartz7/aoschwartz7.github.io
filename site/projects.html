Global prediction of unreported SARS-CoV2 infection from observed COVID-19 casesFall 2020 - Spring 2022-

Many Covid-19 infections remain underreported due to mildness of symptoms and/or inadequate testing. Carson Chow’s lab at the NIH along with Rick Gerkin of Arizona State University used Bayesian inference and a modified SIR model to predict the number of unreported infected individuals from reported cases, recoveries, and deaths. I joined the project to further develop the code base and continue testing modeling developments and public health advancements (vaccines, testing, etc). To showcase US state projections, I built an R Shiny app.

Identifying Snakes in Photos Using Machine LearningJuly - November 2020

Professor Scott Boback of Dickinson College studies patterns in rattlesnake basking activity among other interests. Using wildlife cameras focused on den entrances, he remotely collects images every 15 minutes from sunup to sundown in the spring and fall. Currently, analysis is quite cumbersome. Each photo is closely examined by eye to see if snakes are present. Some are snake-less. Then some reveal piles of camouflaged snakes. If the examiner spots one, they manually annotate it with a small yellow arrow. Finally, these annotations get compiled to track frequency counts and time of day measurements.

I wrote a collection of scripts that use a custom object detection model for automating this process. Technology I used includes: YOLOv3, TensorFlow, Keras, LabelIMG, ImageAI. Code for this project uses Google Colab for access to Google’s GPUs. The dataset for model training can be found here on GitHub.
