Identifying Snakes in Photos Using Machine LearningJuly 2020--Present

Professor Scott Boback of Dickinson College is studying patterns in rattlesnake basking activity. Using wildlife cameras focused on den entrances, he remotely collects images every 15 minutes from sunup to sundown in the spring and fall. Currently, analysis is quite cumbersome. Each photo is closely examined by eye to see if snakes are present. Some are snake-less. Some reveal piles of camouflaged snakes. If the examiner spots one, they manually annotate it with a small, yellow arrow. Finally, these annotations get compiled to track frequency counts and time of day measurements.

I am using machine learning to build a custom object detection model for automating this process. Technology I am using includes: TensorFlow, Keras, LabelIMG, ImageAI, and the object detection system YOLO. Code for this project uses Google Colab for access to Google’s GPUs. The dataset for model training can be found here on GitHub.

NIH NIDDK Covid-19 Open-source ContributionJuly 2020

Many Covid-19 infections remain unreported due to mildness of symptoms and/or inadequate testing. Carson Chow’s lab at the NIH along with Rick Gerkin of Arizona State University are using Bayesian inference and a modified SIR model to predict the number of unreported infected individuals from reported cases, recoveries, and deaths. Their code uses a suite of models of varying complexity to calculate these numbers for different regions,and then produces an average result across models for each region. For a preprint of their paper, click here .

My task was to produce an average result for a user-defined super-region, like “Asia”, “Southern Asia”, or the United States, for example. I familiarized myself with Pandas, using it to parse information on world countries into a format useful for the researchers’ existing scripts. I then modified their existing code to include a parameter for selecting a super-region and extending the results to region-indexed tables.

Twilio RiddlerMarch 2020

This Python-Flask application sends a user an SMS of a random riddle, listens for their response via a Twilio webhook, and sends either the answer or a new riddle. It was built using Flask architecture and Twilio’s Programmable SMS API and Python helper library.
